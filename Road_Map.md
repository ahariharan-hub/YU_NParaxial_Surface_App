Stage 1 — 2D paraxial thin lens

&#x20;   Current V1/V2 foundation.

&#x20;   y-u rays, thin lens matrix, system matrix, Gaussian/cardinal quantities.



Stage 2 — 2D paraxial thick lens / vertex-plane model

&#x20;   Still paraxial.

&#x20;   Use vertex planes and surface matrices.

&#x20;   Good bridge from thin-lens systems to real prescriptions.



Stage 3 — 2D exact Snell meridional tracing

&#x20;   Exact ray-surface intersection.

&#x20;   Exact surface normal.

&#x20;   Exact vector/scalar Snell in the meridional y-z plane.

&#x20;   This is where spherical aberration becomes physically visible.



Stage 4 — 3D paraxial thin lens

&#x20;   Ray vector becomes \[x; y; u; v].

&#x20;   Thin lens acts on both x/u and y/v.

&#x20;   Still first order, but now sagittal/tangential structure can be introduced.



Stage 5 — 3D paraxial thick lens / vertex-plane model

&#x20;   3D first-order surface/translation chain.

&#x20;   Still matrix-based and fast.

&#x20;   Better preparation for real lens layouts.



Stage 6 — 3D exact Snell tracing

&#x20;   Full vector ray-surface intersection.

&#x20;   Full 3D surface normal.

&#x20;   Exact vector Snell.

&#x20;   Basis for real spot diagrams, OPD, wavefront, aberration fitting.



Stage 7 — Aberration layer

&#x20;   Seidel/Zernike/OPD/ray-fan/spot diagnostics.

&#x20;   Should be script/workflow based first.



Stage 8 — ASM / physical optics layer

&#x20;   Scalar field propagation.

&#x20;   Pupil phase from ray trace can feed ASM later.
